# Library Management System
SQL-based Library Management System to manage books, members, borrowing history, due dates, and fine calculations.

1. Project Overview
This project implements a SQL-based Library Management System that manages books, members, and borrowing records.

2. Features
- One-to-many relationships
- Borrowing history tracking
- Due date calculation using DATE_ADD
- Overdue tracking using DATEDIFF
- Fine calculation logic

3. Technologies Used
- MySQL
